Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Отправить код

На электронную почту пользователя отправляется код подтверждения

Example request:
curl --request POST \
    "http://localhost:8000/api/auth/code/send" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"subject\": \"rex16@example.com\"
}"

const url = new URL(
    "http://localhost:8000/api/auth/code/send"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "subject": "rex16@example.com"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "data": null,
    "message": "OK"
}

Tip: Check that you're properly connected to the network.
If you're a maintainer of ths API, verify that your API is running and you've enabled CORS.
You can check the Dev Tools console for debugging information.

    \"birthday\": \"2024-05-30T08:07:16\",

const url = new URL(
    "http://localhost:8000/api/auth/register"

const headers = {
    "Content-Type": "application/json",

Изменить пароль

Эндпоинт необходим для изменения пароля

Example request:
curl --request POST \
    "http://localhost:8000/api/auth/forgot" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"haley.demetris@example.net\",
    \"code\": \"1909\",
    \"password\": \"b-?*`i\'(Ed{$u>=sh{\",
    \"password_confirmation\": \"btrsfnxexzcwejjmiijplyilvifieqvvltluvmndalltfwnprvg\"
}"

const url = new URL(
    "http://localhost:8000/api/auth/forgot"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "haley.demetris@example.net",
    "code": "1909",
    "password": "b-?*`i'(Ed{$u>=sh{",
    "password_confirmation": "btrsfnxexzcwejjmiijplyilvifieqvvltluvmndalltfwnprvg"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/auth/forgot

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

email   string   

Must be a valid email address. Example: haley.demetris@example.net

code   string   

Must match the regex /^([0-9]*)$/. Must be 4 digits. Example: 1909

This field is required when password_confirmation is present. Must be at least 6 characters. Must not be greater than 255 characters. Example: b-?*i'(Ed{$u>=sh{`

This field is required when password is present. The value and password must match. Must be at least 6 characters. Example: btrsfnxexzcwejjmiijplyilvifieqvvltluvmndalltfwnprvg
